Query Selector

  • query selector:allows users to select and retrieve elements from the DOM using CSS selectors.
  • It returns the first element that matches the specified selector
  • var element = document.querySelector(selector);
  • document.querySelector("#myId") selects the element with the ID "myId".
  • document.querySelector(".myClass") selects the first element with the class "myClass".
  • document.querySelector("div.myClass") selects the first
    element with the class "myClass".
  • querySelector returns null if no matching element is found in the document.

Add Event Listener

  • addEventListener method is used to attach an event listener to an HTML element
  • This method allows you to listen for specific events for specific elements, such as mouse click,keypress or form submission.
  • Execute a callback function in response to that event
  • element.addEventListener(event,function,useCapture)
    • element: The HTML element to which you want to attach event listener
    • event: A string represents the name of the event to listen(e.g.,"click","submit")
    • function: The function to be executed when the event is triggered
    • useCapture: A boolean value that specifies whether the event should be executed in the capturing or bubbling phase. It is optional and its default value is false.
// Get the button element
const button = document.getElementById('myButton');

// Add an event listener for the click event
button.addEventListener('click', function(event) {
    console.log('Button clicked!');
});

In Javascript, the event parameter in the function is referring to the event object that contains information about the event that has occurred. When an event is triggered (such as mouse click,keypress or form submission), an event object is automatically created by the browser and passed as an argument to the event handler function


The event object provides various properties and methods that allows you to access information about the event, such as the type of event, the target element that triggered it, the coordinates of the mouse pointer that clicks it and so on.

  • Various properties of event handler are
  • event.type
  • event.key
    • and so on

getElementById

  • Used to retrieve html element from document by its unique identifier known as ID
  • document.getElementById(id)
  • document: This refers to the entire HTML document. It represents the root node of the HTML document tree.
  • getElementById: This is the method used to retrieve an element from the document based on its ID.
  • id: This is a string parameter that specifies the ID attribute value of the element you want to retrieve.
  • querySelector can do every work which can be done from getElementById
